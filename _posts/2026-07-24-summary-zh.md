## [Anthropic 发布 Claude Opus 5，无需数据保留](https://www.anthropic.com/news/claude-opus-5) ⭐️ 9.0/10

Anthropic 发布了 Claude Opus 5，这是一个强大的新 AI 模型，与之前的 Opus 模型一致，不要求对通用访问进行数据保留。该模型的系统卡以 190 页 PDF 形式提供。 这次发布意义重大，因为它提供了一种高性能模型，且没有像 Fable 这样的竞争模型所要求的 30 天数据保留政策。它还引发了关于模型路由的讨论，因为公司越来越需要将任务匹配到最佳模型。 Claude Opus 5 延续了其前身 Claude 4.8 的许多风格特征，一些用户称之为“Claude-isms”。早期测试表明，Opus 5 在图像转 HTML 任务上表现优于 Fable。

## [编译器将计算图转换为标准 Transformer 权重](https://www.reddit.com/r/MachineLearning/comments/1v5fxbe/i_built_a_compiler_that_turns_computation_graphs/) ⭐️ 9.0/10

一款名为 Torchwright 的新型编译器能够将用 Python 编写的任意计算图，转换为标准 Phi-3 Transformer 架构的权重，可直接加载到 Hugging Face 中，无需任何训练或自定义代码。 这项工作通过手工构建的权重使 Transformer 能够直接执行任意算法，与学习行为分离，从而推动了算法推理和机械可解释性发展，并且采用标准架构，便于社区广泛使用。 该编译器针对微软的 Phi-3 模型架构，生成可用标准 Hugging Face 代码加载的检查点。它超越了 RASP 和 Tracr 等先前方法，能够接受普通 Python 计算图，并输出与广泛使用的架构兼容的权重，无需任何自定义组件。

reddit · r/MachineLearning · /u/notforrob · 7月24日 16:15

**背景**: RASP 是一种编程语言，旨在以直接映射到 Transformer 子层的形式表达算法；Tracr 是一个编译器，可将 RASP 程序转换为实际的 Transformer 权重。然而，这些先前工具需要学习 RASP，并且生成的权重针对自定义架构，限制了实用性。Torchwright 扩展了这些想法，允许使用普通 Python 代码作为输入，并针对兼容 Hugging Face 的标准现成 Transformer 架构（Phi-3）。

## [安防摄像头出厂固件含硬编码 GitHub 管理员令牌](https://hhh.hn/hanwha-github-token/) ⭐️ 8.0/10

一家安防摄像头厂商在出厂固件中嵌入了具有管理员权限的 GitHub 个人访问令牌，该令牌被发现硬编码在登录页面源代码中。此令牌可能允许未经授权访问该厂商的 GitHub 仓库。 该漏洞暴露了一个严重的供应链安全缺陷，因为发现该令牌的攻击者可获得对厂商代码仓库的完全访问权限，从而可能在固件更新中注入恶意代码。这凸显了物联网设备制造中安全凭证管理的迫切需求。 硬编码的 GitHub 令牌出现在摄像头网页界面的登录页面中，赋予了对厂商 GitHub 账户的管理员访问权限。此类令牌绝不应包含在出厂固件中；它们会导致仓库被入侵并引发进一步攻击。

hackernews · hhh · 7月24日 11:54 · [社区讨论](https://news.ycombinator.com/item?id=49034292)

**背景**: GitHub 个人访问令牌（PAT）用于通过 API 或命令行与 GitHub 交互时替代密码进行身份验证。硬编码凭证（包括嵌入固件中的 API 令牌）是众所周知的安全反模式，因为它们静态不变、广泛分发且常被遗忘，成为攻击者的主要目标。此次事件延续了嵌入式设备漏洞的模式，即秘密被无意中暴露在用户可访问的界面中。

## [英伟达、微软、Meta 呼吁谨慎监管开放权重 AI 模型](https://www.cnbc.com/2026/07/24/nvidia-microsoft-meta-open-weight-ai-models.html) ⭐️ 8.0/10

英伟达、微软和 Meta 联合发布公开信，警告过度监管开放权重 AI 模型可能损害美国在人工智能领域的领先地位，并敦促政策制定者避免采取限制性措施。 这三大科技巨头的联合立场可能对美国 AI 监管辩论产生重大影响，凸显了开放权重支持者与主张加强管控者之间的分歧日益扩大，并强调了开放模型在全球 AI 竞争中的战略重要性。 该信函特别指出，开放权重模型促进创新和安全研究，而过度监管将把研发推向海外，尤其是中国。社区评论引用了一个相关的 HN 讨论串，其中初创公司创始人敦促美国不要切断中国的开放权重 AI。
